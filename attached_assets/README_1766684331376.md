## üõ†Ô∏è Technology Stack

### Backend
- **Framework:** FastAPI 0.104+
- **Language:** Python 3.11+
- **Database:** PostgreSQL 15, MongoDB 6
- **Cache:** Redis 7
- **Message Queue:** RabbitMQ 3.12
- **NLU:** Transformers, spaCy, Rasa
- **LLM:** OpenAI GPT-4, Anthropic Claude

### Frontend
- **Framework:** React 18 with TypeScript
- **UI Library:** Material-UI 5
- **State Management:** Redux Toolkit
- **Charts:** Recharts
- **WebSocket:** Socket.io

### Infrastructure
- **Container:** Docker 24+
- **Orchestration:** Kubernetes 1.24+
- **Monitoring:** Prometheus, Grafana
- **Logging:** ELK Stack
- **Tracing:** Jaeger

## üöÄ Quick Start

### Local Development

```bash
# Clone repository
git clone https://github.com/yourorg/b2t-voice.git
cd b2t-voice

# Start all services
docker-compose up -d

# Run database migrations
docker-compose exec api alembic upgrade head

# Seed initial data
docker-compose exec api python scripts/seed_data.py

# Access the platform
# Frontend: http://localhost:3000
# API: http://localhost:8000
# API Docs: http://localhost:8000/docs
```

### Manual Setup

#### Backend Setup

```bash
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set environment variables
cp .env.example .env
# Edit .env with your configuration

# Run migrations
alembic upgrade head

# Start server
uvicorn app.main:app --reload
```

#### Frontend Setup

```bash
cd frontend

# Install dependencies
npm install

# Set environment variables
cp .env.example .env.local
# Edit .env.local with your configuration

# Start development server
npm start
```

## üîß Core Components

### 1. B2T Mini VoiceAgents

**Purpose:** Rapid deployment of pre-configured conversational agents

**Key Features:**
- Template-based agent creation
- Configuration-driven setup
- One-click deployment
- Version management

**Usage Example:**
```python
from services.agent_factory import AgentFactory

# Create agent from template
agent = await agent_factory.create_agent(
    template_id="banking_assistant_v1",
    name="Customer Support Bot",
    user_config={
        "integrations": {
            "core_banking": {
                "baseUrl": "https://api.bank.com"
            }
        }
    }
)
```

### 2. NLU Engine

**Purpose:** Natural language understanding with intent classification and entity extraction

**Key Features:**
- Multi-model support (Transformers, Rasa, LLM)
- Custom entity extraction
- Sentiment analysis
- Context management

**Usage Example:**
```python
from services.nlu_service import CompositeNLUService

# Process user input
result = await nlu_service.process(
    text="Transfer $500 to savings",
    agent_id="agent_123",
    session_id="session_456"
)

# Result contains:
# - intent: "transfer_money"
# - entities: {"amount": 500, "to_account": "savings"}
# - confidence: 0.95
```

### 3. Orchestrator

**Purpose:** Manage dialog flows and business logic

**Key Features:**
- Visual workflow design
- State machine management
- Multi-turn conversations
- Integration routing

**Usage Example:**
```python
from services.orchestrator import OrchestratorEngine

# Execute workflow
result = await orchestrator.execute_workflow(
    workflow_id="money_transfer_flow",
    nlu_result=nlu_result,
    session_id="session_456",
    user_id="user_789"
)
```

### 4. Integrations

**Purpose:** Connect to external services and APIs

**Key Features:**
- REST, GraphQL, WebSocket support
- OAuth 2.0, API Key authentication
- Retry logic and rate limiting
- Pre-built connectors

**Usage Example:**
```python
from services.integrations.connectors.salesforce import SalesforceIntegration

# Create Salesforce integration
sf = SalesforceIntegration(
    instance_url="https://mycompany.salesforce.com",
    client_id="...",
    client_secret="..."
)

await sf.connect()
lead_id = await sf.create_lead({
    "FirstName": "John",
    "LastName": "Doe",
    "Company": "Acme Corp"
})
```

### 5. Insights & Analytics

**Purpose:** Monitor performance and gather business intelligence

**Key Features:**
- Real-time dashboards
- Conversation analytics
- NLU performance metrics
- Business KPIs

**Usage Example:**
```python
from services.analytics.conversation_analytics import ConversationAnalytics

# Get metrics
metrics = await analytics.get_conversation_metrics(
    agent_id="agent_123",
    start_date=datetime.now() - timedelta(days=30),
    end_date=datetime.now()
)

# Returns:
# - total_conversations
# - avg_conversation_length
# - completion_rate
# - avg_response_time
```

```typescript
// Redux slice example
import { createSlice, createAsyncThunk } from '@reduxjs/toolkit';

export const fetchAgents = createAsyncThunk(
  'agents/fetchAll',
  async () => {
    const response = await agentsAPI.list();
    return response.data;
  }
);

const agentsSlice = createSlice({
  name: 'agents',
  initialState: { items: [], loading: false },
  extraReducers: (builder) => {
    builder
      .addCase(fetchAgents.pending, (state) => {
        state.loading = true;
      })
      .addCase(fetchAgents.fulfilled, (state, action) => {
        state.items = action.payload;
        state.loading = false;
      });
  }
});
```
